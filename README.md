# CodeScanner
**The iOS application for scanning QR codes and barcodes.**

### Technology stack:
**SwiftUI, MVVM, CoreData, AVFoundation, URLSession, Combine,
Swift Concurrency.**

Using [Open Food Facts API](https://world.openfoodfacts.org) to fetch product information.
